# Financial Risk Analysis using Linear Algebra

## Overview

This project models financial relationships between major banks using linear algebra.
It analyzes how a shock in one bank can affect others using matrices and eigenvalues.


## What I Did

* Represented bank interdependencies using a matrix
* Computed eigenvalues and eigenvectors using NumPy
* Verified Cayley-Hamilton theorem
* Simulated market shock and analyzed its impact
* Visualized results using graphs


## Files

* financial_market_risk_analysis.ipynb – code and visualizations
* financial_risk_analysis_linear_algebra.pdf – detailed report


## Tools Used

* Python
* NumPy
* Matplotlib
* seaborn
* Google Colab


## Key Insight

A shock in one bank propagates through the system due to interconnections, and eigenvalues help identify major risk patterns.
